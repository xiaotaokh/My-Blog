Markdown 扩展示例 ​

本页演示了 VitePress 提供的一些内置 markdown 扩展。

语法高亮 ​

VitePress 提供由Shiki支持的语法高亮功能，以及行高亮等附加功能：

输入

md
```js{4}
export default {
  data () {
    return {
      msg: 'Highlighted!'
    }
  }
}
```

输出

js
export default {
  data () {
    return {
      msg: 'Highlighted!'
    }
  }
}

输入

md
::: info 信息
这是一个信息框。
:::

::: tip 提示
